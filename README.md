# 使用方法

目前仅支持windows用户使用。

1. 安装好g++并添加环境变量
2. 安装好python并添加环境变量
3. 双击运行主目录下的`First.bat`
4. 在主目录下的`config.yml`中按照提示填写
5. 运行`Second.bat`，如果电脑是32位运行`Second-32.bat`，在result.txt中得到最终markdown源码

注意！从第二次开始使用起即可直接从第三步开始。
